v1.2.0LatestStable
March 2026
Command Palette · XP Gamification · 12 Templates · Settings Panel
The biggest single update — a VS Code-style palette, earn XP for completing tasks, quick-start templates, and full settings.
+
⌘ Command Palette
  • Ctrl+K opens palette from anywhere in the app
  • 27 commands: views, themes, actions, navigation
  • Live task search as you type
  • Keyboard: arrow keys + Enter, Esc to close
🎮 XP and Levels
  • Complete tasks to earn XP (10–50 per task by priority)
  • Recurring tasks give +20% bonus XP
  • 11 levels from Beginner to Grandmaster
  • Level-up banner animation on level increase
  • XP progress bar in sidebar
  • Skill tag field tracks XP per skill area
📋 Task Templates (12)
  • Morning Workout, Study Session, Code Review
  • Daily Standup, Weekly Review, YouTube Upload
  • Meditation, Reading, Run, Report, Ideas, Bills
  • One click to add with all fields pre-filled
⚙️ Settings Panel
  • Sound toggle and volume slider
  • Peak Focus Hours with live banner in Today view
  • XP system and level-up notification toggles
  • Export buttons, storage usage display
  • Clear all data option with confirmation
🔧 Architecture
  • Full task engine extracted to standalone app.js (67KB)
  • IIFE pattern — clean global scope management
  • New Skills and XP view in sidebar
  • Peak hours shown as ⚡ on task time badges
